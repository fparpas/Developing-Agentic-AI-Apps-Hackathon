# Challenge 00 - Set Up and Prepare Your Environment

**[Home](../README.md)** - [Next Challenge >](./Challenge-01.md)

## Introduction

In this session, you will set up your computer and cloud environment with the minimum required tools.

## Pre-requisites

- Your laptop (development machine): Windows, macOS, or Linux on which you have **administrator rights**.
- Active Azure Subscription with **Owner access** to create and modify resources, and manage role assignments
- Access to Azure OpenAI in the desired Azure subscription.
- Latest version of Azure CLI
- GitHub Copilot (free tier with limitations available, or start a Copilot Pro trial for enhanced features)
- For C#/.NET track:
  - Latest version of Visual Studio Code
  - .NET 8.0 SDK or later version
- For Python track:
  - Latest version of Visual Studio Code with the Python extension and Jupyter package
  - Python 3.12 or later version

## Description

Set up and configure the following tools:

- Use your active Azure Subscription or the one provided for the hackathon.
- Log into the [Azure Portal](https://portal.azure.com) and confirm that you have an active subscription to which you can deploy cloud services.
- Set up [GitHub Copilot](https://github.com/features/copilot/plans) – if you don't have a license, start a Copilot Pro trial for enhanced AI assistance during development, or use the free tier with limited features.
- Install the latest version of [Visual Studio Code](https://code.visualstudio.com) if you don't already have it installed.
- Clone this GitHub repository onto your workstation.
- For C#/.NET track:
  - Install .NET 8.0 SDK or later version
- For Python track:
  - Install Python 3.12 or later version
  - In Visual Studio Code, install the [Python extension](https://marketplace.visualstudio.com/items?itemName=ms-python.python) and the [Jupyter package](https://pypi.org/project/jupyter/)


## Success Criteria

- ✅ You can log in to the Azure Portal.
- ✅ You have a Bash shell or PowerShell available (you can also use Azure Cloud Shell).
- ✅ Running `az --version` shows the installed version of the Azure CLI.
- ✅ Visual Studio Code is installed.
- ✅ .NET 8.0 SDK or Python is installed (verify with `dotnet --version` or `python --version`).
- ✅ You have access to GitHub Copilot.
- ✅ You have cloned the GitHub repository onto your workstation.
